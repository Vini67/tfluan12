TF - Frontend — Aula 12
Chat em Tempo Real com WebSocket, RabbitMQ e Reações Rápidas

Este projeto implementa um sistema de chat em tempo real utilizando WebSocket integrado ao RabbitMQ (Exchange Fanout).
Na Aula 12, foi adicionada a funcionalidade de reações rápidas, permitindo que o usuário envie reações pré-definidas no chat.
Essas reações seguem o mesmo fluxo das mensagens normais: WebSocket → RabbitMQ → WebSocket → Interface React.
